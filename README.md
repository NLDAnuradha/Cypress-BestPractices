## 📝 Key Cypress Commands Learned

```javascript
// Basic commands
cy.visit('url')
cy.get('selector')
cy.click()
cy.type()
cy.should()

// Assertions
cy.should('be.visible')
cy.should('contain', 'text')
cy.should('have.attr', 'attribute')

// Navigation
cy.go('back')
cy.go('forward')
cy.reload()

// Waiting
cy.wait()
cy.intercept()
```
